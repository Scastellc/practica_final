
Este es un fichero que contiene una pequeña explicación de cada carpeta/fichero que hay en este repositorio.

Chessline.tk es el dominio donde se aloja la pagina.

La aplicación funciona al 100% en Google Chrome y también en Mozilla.

Aquí veremos los diagramas y documentación utilizados y la base de datos, mas un diagrama sin extensión que se puede abrir con pain y es el diagrama que me genero phpMyAdmin sobre la base de datos.

Dentro de la carpeta practica_final encontraremos:
Carpetas:
	Css se encuentran todas las hojas de estilo.
	Pieza tiene las imágenes de las piezas del tablero.
	Noticias tiene todas las imágenes de las noticias subidas en la pagina noticias.
	Js tiene todos los Scripts de JavaScript, las funciones y el fichero necesario para utilizar JQuery.
	Intro se encuentra las imágenes que aparecen en el index.
	Php se encuentra todo lo relacionado con php, la clase usuario creada, las funciones, la conexión con la base de datos, y la función para quitar la sesión.
	Usuario para almacenar los avatares de los usuarios

Ficheros:
	Index es el inicio de la pagina.
	Partida es donde ahora se encuentra el tablero y se puede "jugar".
	Login para acceder como usuario registrado.
	ModificarCuenta para poder modificar la sesión del usuario 
	Noticias para ver las noticias, las voy añadiendo de forma manual para seleccionar mejor las noticias que deseo mostrar.
	Formulario para registrar nuevos usuarios.
